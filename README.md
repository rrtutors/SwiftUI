# SwiftUI
Some of Xcode shortcut keys for developers

    Cmd + Click: Contextual menu for code or UI
    Option + Click: Quick info for code
    Cmd + 0: Show/hide Navigator
    Cmd + Shift + L: Insert new element
    Cmd + R: Run the app
    Cmd + . : Stop the app
